## Demo Document Command

The repository includes fake demo documents under `company_docs/`, including:

```text
company_docs/billing.md
company_docs/demo_billing.md
```

To recreate the smaller demo document manually:

```bash
echo "# Demo Billing Migration Evidence" > company_docs/demo_billing.md
```

The demo content must remain fake and include:

- one supported billing migration claim
- one risk claim
- one `CONTRADICTS:` marker
